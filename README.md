## Project Overview





This project is a complete SQL-based data analysis case study built using a real-world Zepto product inventory dataset. The objective was to simulate how a Data Analyst works with retail and e-commerce data to uncover business insights related to pricing, inventory management, discount strategies, and product performance.



The project covers the complete analytics workflow:



Data Exploration



Data Cleaning



Business KPI Analysis



Inventory \& Pricing Insights



SQL-Based Decision Making



This project was developed using \*\*MySQL\*\* and focuses on solving practical business problems through SQL queries.



\---



## Dataset Overview



The dataset contains 3,000+ product records (SKUs) from Zepto across multiple categories including:



\* Fruits \& Vegetables

\* Dairy \& Breakfast

\* Snacks \& Munchies

\* Beverages

\* Packaged Foods

\* Personal Care

\* Household Essentials



Each row represents a unique product with information related to pricing, discounts, stock availability, quantity, and weight.



\### Dataset Columns



\* Category

\* name

\* mrp

\* discount\_percent

\* available\_quantity

\* discounted\_selling\_price

\* weight\_in\_gms

\* out\_of\_stock

\* quantity
